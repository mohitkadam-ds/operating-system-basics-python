Python programs based on Operating System practicals covering system information, directory structures, and file and folder operations.
# Operating System Basics using Python
This repository contains Python programs based on Operating System
practical assignments related to system information and file system
management.
## Practicals Included
- Display operating system name and version
- Retrieve system and hardware information
- Representation of directory structures
- Folder creation using OS module
- Department-wise directory and file creation
